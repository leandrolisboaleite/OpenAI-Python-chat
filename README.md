# Simple chat for OpenAI

A simple chat to use openAI on your terminal :)



## Installation

Install dependencies

```bash
pip3 install -r requirements.txt
```
    
Add your API Key to bashrc

```bash
  vi ~/.bashrc
  export OPENAI_API_KEY="KEY"
  source ~/.bashrc
```
